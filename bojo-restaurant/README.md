# BOJO Restaurant Website

A modern, responsive restaurant website for BOJO featuring a photo gallery, menu ordering system, and elegant design.

## Features

- **Modern Design**: Clean, elegant interface with smooth animations
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Photo Gallery**: Interactive gallery with lightbox functionality
- **Menu Ordering**: Functional ordering system with cart management
- **Navigation**: Smooth single-page application navigation
- **Mobile-Friendly**: Hamburger menu for mobile devices

## Pages

1. **Home**: Hero section with call-to-action
2. **About Us**: Restaurant story and features
3. **Menu**: Interactive menu with categories (Appetizers, Main Courses, Desserts, Beverages)
4. **Gallery**: Photo gallery showcasing restaurant and dishes

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with animations and transitions
- **JavaScript**: Interactive functionality and DOM manipulation
- **Google Fonts**: Playfair Display and Lato for typography
- **Picsum Photos**: Placeholder images for gallery

## How to Use

1. Open `index.html` in a web browser
2. Navigate between sections using the navigation menu
3. Browse the menu and add items to your order
4. View the photo gallery with lightbox functionality
5. Use the mobile hamburger menu on smaller screens

## Features Details

### Navigation
- Fixed navigation bar with smooth scrolling
- Active state indicators
- Mobile-responsive hamburger menu

### Menu System
- Category-based menu organization
- Add to cart functionality
- Order modal with cart management
- Price calculation

### Gallery
- Grid layout with hover effects
- Lightbox viewer for full-size images
- Keyboard navigation (arrow keys, escape)
- Touch-friendly on mobile devices

### Responsive Design
- Mobile-first approach
- Flexible grid layouts
- Optimized for all screen sizes

## Customization

### Colors
- Primary color: `#ff6b6b` (coral red)
- Background: `#f8f8f8` (light gray)
- Text: `#333` (dark gray)

### Fonts
- Headings: 'Playfair Display' (serif)
- Body text: 'Lato' (sans-serif)

### Images
Replace the placeholder images from Picsum Photos with your own restaurant photos:
- Hero background image
- Gallery images
- Update image URLs in the HTML file
